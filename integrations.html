KISSmetrics offers the ability to add data from other external data sources into KISSmetrics.

If your platform is not listed, please see if your platform has a data-out API - for example, Twitter and Shopify provide APIs for retrieving data. If you can get data out of your platform, then you can potentially import it back into KISSmetrics.
